# Api_Valoriza

Projeto Valoriza

- Cadastro de Usuario;
- Cadastro de Tags;
- Cadastro de Elogios;
- Listagem de Usuarios;
- Listagem de Elogios enviados;
- Listagem de Elogios Recebidos;

* Autenticação JWT;

Framework usado (Node.js);

Bibliotecas usadas:
  -> typescript
  -> express
  -> typeORM
  -> JWT
  -> bcryptjs
  -> class-transformer
